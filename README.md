## Python內建三種資料結構
1. list:串列資料 [x,x,x,x] 有順序之分，索引號由左而右從0號開始。<br />
   tuple: (x,x,x,x) 跟list很像的資料結構，差別在tuple是暫時性的資料結構，不能修改元素。簡易寫法可不加小括號()。<br />
   for 自訂變數 in 串列資料 <br />
2. dict:對應資料 
3. set
